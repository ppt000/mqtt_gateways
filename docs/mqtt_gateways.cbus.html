Lights dictionary. Derived from database. The key is the internal name for the lights (in this case it was the latest version of the name in the database, but it should not be assumed that it will stay the same). The value is a pair of strings, the first one being the address of the light in the C-Bus network, and the second one being the internal name of the location where this light is.
