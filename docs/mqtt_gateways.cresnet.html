mqtt_gateways.cresnet.cresnet_data module¶

Created on 4 Jan 2017 Module containing all the codes to control the CRESTRON system, and their representation inside this package. @author: PierPaolo

mqtt_gateways.cresnet.cresnet_data.ACTIONS = {'POWER_ON': <function <lambda>>, 'SOURCE_TUNER': <function <lambda>>, 'SOURCE_CD': <function <lambda>>, 'MUTE_TGL': <function <lambda>>, 'POWER_OFF': <function <lambda>>, 'VOL_DN': <function <lambda>>, 'LIGHT_OFF': <function <lambda>>, 'SOURCE_SKY': <function <lambda>>, 'MUTE_OFF': <function <lambda>>, 'MUTE_ON': <function <lambda>>, 'SOURCE_IPOD': <function <lambda>>, 'SOURCE_DVD': <function <lambda>>, 'VOL_UP': <function <lambda>>, 'LIGHT_ON': <function <lambda>>}¶

Dictionary of all Keypresses as they arrive from CRESNET. The respective values are a list with 2 elements, the room name (as defined in the ROOMS list) and a command name (as defined in the ACTIONS dictionary).

mqtt_gateways.cresnet.cresnet_data.DEVICES = [['\x03', 'Adagio_Unit'], ['\x08', 'Kitchen_Keypad'], ['\t', 'DiningRm_Keypad'], ['\n', 'Office_Keypad']]¶

Dictionary of sources available on the Adagio unit, with their respective ‘id’ in CRESNET. Format is: Internal name as a string, input on Adagio in hex code exactly as sent over the serial interface.

mqtt_gateways.cresnet.cresnet_data.FUNCTIONS = ['Lighting', 'AudioVideo']¶

Dictionary of rooms available to operate with this gateway. The format is: { room name in string form: list of Adagio channels that output sound to this room (in hex code form to be sent directly to CRESNET) }

mqtt_gateways.cresnet.cresnet_data.ROOMS = {'DiningRm': ['\x02'], 'MasterBath': ['\x05'], 'Terrace': ['\x06'], 'Office': ['\x01'], 'Kitchen': ['\x03', '\x04']}¶

List of devices available on CRESNET. The key is the ‘address’ of the device expressed as hex code exactly as it is used in the CRESNET protocol. The value is a string description or name of the device, only used for logging purposes for now.

mqtt_gateways.cresnet.cresnet_data.SOURCES = {'NONE': '\x00', 'DVD': '\x08', 'IPOD': '\x01', 'SKY': '\x06', 'CD': '\x04', 'TUNER': '\x03'}¶

Dictionary of all the commands available on this gateway, with their respective lambda function definition.
